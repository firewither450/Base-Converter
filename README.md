# Base-Converter
Contains python functions for conversions from and to any base from base 2 to the UNICODE character limit (71859 characters) 

** THIS CODE HAS NOT BEEN RIGOROUSLY TESTED AND CAN CONTAIN ERRORS ** 

This contains a the definition of a Num object that has a function  base(n: int) which converts the number to the desired base n. If no parameters are given, it will convert to base 10.
When initialised, input the number in any base as a string and the base as an integer.
To get the max amount of characters, set the parameter as 0.
It still needs plenty of optimisation for larger bases.

This is a personal project to test my python knowledge and skill, I am aware there are easier ways to achive this.
